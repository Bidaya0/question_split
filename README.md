# 在线答题系统

## 主要功能列表

1. 学习模式
   - 考试模拟模式：随机抽取100道题进行模拟考试
   - 背题模式：持续随机抽取题目进行练习

2. 错题管理
   - 自动记录错题
   - 错题回放功能
   - 错题汇总显示

3. 答题功能
   - 单选题答题
   - 实时显示答案和解析
   - 进度显示（考试模式）

4. 结果统计
   - 显示总题数
   - 显示正确题数
   - 显示错误题数
   - 错题题号列表
